# Slack Badger App

A Slack app for gamifying expertise and engagement within your workspace. Users can create badges, ask questions, tag experts, award best answers, and track leaderboards.

## Features

- Create and manage badges (`/create-badge`, `/delete-badge`, `/list-badges`)  
- Ask questions tagged to a badge (`/ask`)  
- Find users who have earned badges (`/find-experts`)  
- Award best answers with points (`/best-answer`)  
- Track weekly and lifetime points  
- View personal badges and stats (`/my-badges`, `/my-stats`)  
- Weekly leaderboard posts
